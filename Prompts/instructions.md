## Overview

I'm hosting a home cafe. I also run a food channel on Instagram with reels which feature items I cook or bake and captions have recipes. This website is
to both serve as a pretend cafe website as if it were an actual cafe, as well
as the invite for it. Later, I want to be able to add pictures, a menu, and
importantly, integrated reels with instructions for how to make each item as I
will make videos for each item I serve. I also want to add recipes directly to
the site. It should be streamlined, professional, but also cozy and modern.

## Features

- Home Page -- short, aesthetic, modern
    - Contains basic pictures of food being served
    - Contains invite (date, location, etc.)
    - Links to other pages
- Menu
    - Split between food and drinks
- Some sort of food item page
    - Optional instagram reel integration
    - Recipe for this item
    - Photos of course
